Argo CD pulls in this chart directly from the repo and applies the appropriate `values-*.yaml` based on the target

To test locally:

```bash
helm upgrade --install rolesui ./rolesui -f values-dev.yaml
```

To remove:

```bash
helm uninstall rolesui
```
